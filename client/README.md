# Simple Chat Application

A real-time chat application built using React, Node.js, Express, and Socket.io.

## Features

- Real-time messaging
- Dummy user login
- Message timestamps
- Responsive UI
- Socket.io communication

## Tech Stack

- React (Vite)
- Node.js
- Express.js
- Socket.io

## Installation

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Frontend:
http://localhost:5173

Backend:
http://localhost:5000
